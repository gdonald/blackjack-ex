## Blackjack

*Console Blackjack in Elixir*

#### Install dependencies

```
mix deps.get
```

#### Run specs

```
mix espec
```

#### Run coverage

```
mix coveralls.html
```

Results will be in ./cover

#### Build

```
mix escript.build
```

#### Run

```
./blackjack
```
